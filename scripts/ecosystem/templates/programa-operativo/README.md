# Programa Operativo — Logística

Gestión de solicitudes logísticas empresariales. Parte del ecosistema [La Sucursal del Café](https://github.com/lasucursaldelcafe-droid).

## Inicio rápido

```bash
npm install
npm run setup
npm run db:init
npm run dev
```

## Ecosistema

- Enlazado desde [Colombia Green Coffee `/logistica`](https://github.com/lasucursaldelcafe-droid/Coffee-hunter-cov2)
- Configuración masiva: `npm run setup:ecosystem` en Coffee-hunter-cov2

## Stack

Next.js 16 · Drizzle · SQLite local / Turso producción · Zod
